# ☕ Coffee Shop Management System (Backend API)

A production-ready, enterprise-grade RESTful API built with Java and Spring Boot to power a modern Coffee Shop ecosystem. This backend handles everything from customer registration and user authentication to product catalogs, dynamic order processing, and delivery addresses.

---

## 🛠️ Tech Stack & Architecture

* **Backend Framework:** Java 17 / Spring Boot 4.x
* **Database:** PostgreSQL (with HikariCP connection pooling)
* **Data Access:** Spring Data JPA & Hibernate ORM
* **Mapping & DTOs:** MapStruct (for high-performance, decoupled object layers)
* **Security (Planned):** Spring Security (Role-Based Access Control: CUSTOMER, BARISTA, ADMIN)
* **Tools & Version Control:** VS Code, Git, GitHub, Postman
